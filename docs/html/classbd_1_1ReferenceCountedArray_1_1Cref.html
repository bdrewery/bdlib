Detailed Description

template<class T, class Allocator = std::allocator<T>>
class bd::ReferenceCountedArray< T, Allocator >::Cref

Safe element reading and writing. This class should be optimized away and fully inlined such that: ReferenceCountedArray s; s[0] = 'a'; Should be rewritten as: s.write(0, 'a');.
