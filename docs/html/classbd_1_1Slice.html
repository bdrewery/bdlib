Detailed Description

template<class T>
class bd::Slice< T >

Safe subarray reading and writing. This class should be optimized away and fully inlined such that: ReferenceCountedArray s("look over there"); s(0, 4) = "blah"'; Should be rewritten as: s.replace(0, "blah", 4);.
