Ensure that the buffer capacity() is >= newSize; else grow/copy into larger buffer.

Parameters
newSizeA size that we need to Allocate the buffer to.
scaling_factorHow much to multiply the size by to help avoid later resizing
offsetThe offset of the old buffer so we know where to start
sublenThe length of the subarray in use
Precondition
newSize is > 0 (assumed as size_t is unsigned)
Postcondition
The buffer is at least nsize bytes long.
If the buffer had to grow, the old data was deep copied into the new buffer and the old deleted.
